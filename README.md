# Basketball Team Generator

Basketball Team Generator is a browser-only pickup basketball app for creating balanced teams from saved player profiles. Add each player’s height, weight, play style, ability tiers, and evolving rating, then select the players available for a game and generate fair randomized teams based on weighted basketball attributes. After a game, enter player stats to update ratings and track cumulative leaderboard totals. The app runs entirely from `index.html` and `app.js`, stores data in local storage, and does not require a backend.
